About This Project

This problem was given to me as part of my Energy Australia graduate interview process.

The data was provided to us via an API, however I found that when I was working on this project, the API would be unavailable so I also used an offline json of the data I would have received as a fallback mechanism.

Another issue I had with the API was that it would give a CORS 'no Access-Control-Allow-Origin' error, so I had hosted a simple Node backend proxy that would forward the request with such header.

Most of this project was completed before my given deadline, however I have worked on it since the deadline, adding stylistic changes as well as filling out this page.

 View on Github The Problem

Your team is tasked with listing out music festival data in a particular manner: at the top level, it should show the band record label, below that it should list out all bands under their management, and below that it should display which festivals they've attended, if any. All entries should be sorted alphabetically.

For example:

  • Record Label 1
    • Band X
      • Omega Festival
    • Band Y
  • Record Label 2
    • Band A
      • Alpha Festival
      • Beta Festival
